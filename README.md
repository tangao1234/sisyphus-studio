# 西西弗斯 · 剧本工作室

四阶流水线（大纲 → 分场 → 剧本 → 镜头）的剧本创作工作台。Fountain 语法实时排版，内置项目快照、三维度剧本体检（结构 / 场景 / 人物弧光，启发式评估）以及项目与全量备份的导入导出。

线上地址：https://tangao1234.github.io/sisyphus-studio/

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

产物输出到 `dist/`，线上版本部署在 `gh-pages` 分支。

## 技术栈

React 19 + TypeScript + Vite 7 + Tailwind CSS v3 + shadcn/ui
